# Roadmap

## Now

- Keep this repo as the final product workspace.
- Preserve the existing prototype under `prototypes/healther-vision`.
- Start production implementation from event schemas and backend skeleton.

## MVP

- 4 to 8 bed pilot.
- Edge ingest and state machine.
- Command center.
- Evidence clips.
- Alert review workflow.
- Vitals OCR.
- Ask AINA for event and patient summaries.
- Morning digest.

## V1

- Single-hospital production deployment.
- Hardened RBAC and audit logs.
- Better model evaluation.
- FHIR-compatible reviewed observations.
- Notification escalation.
- Deployment runbooks.

## V2

- Multi-bed ward optimization.
- Better edge models.
- DICOMweb imaging retrieval.
- Deeper EMR integrations.
- Mobile tablet app.
- Advanced clinical validation studies.

## Deliberately Later

- Predictive deterioration.
- rPPG/contactless vitals.
- Autonomous treatment workflows.
- Cross-camera identity tracking.
